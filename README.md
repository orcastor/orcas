主要特性：

- ✅ 对象级重删（秒传）
- ✅ 小对象打包
- ✅ 大对象分片
- ✅ 对象多版本
- ✅ 零知识加密（端到端加密，国际标准算法）
- ✅ 智能压缩
- [ ] [差分同步](https://github.com/orcastor/xdelta)
- [ ] 实时快照
